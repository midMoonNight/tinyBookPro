# 极简记账本 Pro

微信小程序 MVP，支持未登录本地记账、日/月统计、本月预算提醒，以及登录后的云端同步入口。

## 使用方式

用微信开发者工具打开本目录即可预览。

当前实现为原生小程序结构：

- `pages/ledger`：记账、今日概览、日/月统计
- `pages/profile`：登录状态、预算设置、数据同步
- `utils`：本地存储、统计计算、云同步辅助

如果未配置微信云开发环境，登录会降级为本地模式，不影响本地记账能力。登录按钮触发后会尝试通过微信授权获取头像和昵称，并在页面只读展示。

## 云同步配置

需要真实云同步时：

1. 在 `project.config.json` 中替换真实 `appid`。
2. 在 `utils/cloud-config.js` 中确认 `CLOUD_ENV_ID` 等于云开发控制台里的环境 ID。
3. 上传 `cloudfunctions/login` 云函数。
4. 在云数据库中创建 `records` 和 `budgets` 集合。

未完成以上配置时，小程序仍可本地记账，但不会显示云同步成功状态。
